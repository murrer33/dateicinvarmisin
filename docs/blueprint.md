# **App Name**: Date Proposal Wizard

## Core Features:

- Interactive Proposal Flow: Guide the user through a series of steps to propose a date.
- Fleeing 'No' Button: Implement a button that moves randomly to prevent the user from clicking 'No'.
- Celebratory Konfetti: Trigger a confetti animation upon acceptance.
- Date Type Selection: Allow the user to select a preferred type of date (coffee, dinner, etc.).
- Location Preference: Allow the user to choose the date location.
- Contact Information Input: Collect contact information to arrange the date.
- Summary Display: Display a summary of the user's choices.

## Style Guidelines:

- Primary color: Soft pink (#F4B4C3) to evoke a romantic feel.
- Background color: Very light pink (#F9E7EA), nearly white, for a gentle and inviting backdrop.
- Accent color: Heart red (#E57373), for highlights, the 'Yes' button, and confetti.
- Font: 'Poppins', a geometric sans-serif, for a contemporary and readable style.
- Card-based UI for each step of the proposal flow.
- Rounded corners for buttons and cards to enhance the cute aesthetic.
- Smooth transitions (fade-in/out) between steps.
